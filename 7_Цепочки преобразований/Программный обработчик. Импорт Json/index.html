Программные обработчики

Импорт JSON

Данная инструкция описывает пример настройки цепочки преобразований, включающей объект "Программный обработчик", предназначенный для импорта данных из JSON файла, размещенного в файловом хранилище.

Шаг 1: Загрузка файла в файловое хранилище

  1. Перейдите в раздел "Файлы".
  2. Загрузите файл формата JSON с данными. Пример файла: ```json

{ "data": [ { "id": 1, "name": "Объект 1" }, { "id": 2, "name": "Объект 2" } ] }```

Корневой объект должен содержать одно поле с массивом объектов (в данном примере — поле data).

Убедитесь, что файл успешно добавлен в файловое хранилище.

Шаг 2: Инициализация обработчика

  1. Перейдите в раздел "Цепочки".
  2. Настройте программный обработчик:
  3. Нажмите на кнопку вызова списка обработчиков.

Выберите "Программный обработчик. Импорт JSON" из списка. Пример интерфейса:

В рабочей области появится объект обработчика в виде блока с синим заголовком.

Шаг 3: Настройка параметров обработчика Скопируйте путь к загруженному JSON файлу из раздела "Файлы":

Нажмите на файл правой кнопкой мыши. Выберите "Копировать путь". Пример интерфейса:

Настройте параметры обработчика в поле params:

json Copy code { "file_path": "путь_к_JSON_файлу.json", "array_field": "data" } file_path: Путь к файлу в файловом хранилище. array_field: Имя поля, содержащего массив объектов (в данном примере — data). Подключите выходную коллекцию к обработчику. Это позволит сохранять импортированные данные в коллекцию.
