Загрузка данных из CSV файлов

  1. Перейдите в раздел Коллекции и нажмите на элемент загрузки над списком коллекций.

2. В диалоговом окне выберите файл CSV. 3. В открывшемся окне, выполните настройки загружаемого файла: - Введите имя коллекции (не должно содержать символов . и $). - Настройте разделитель, чтобы привести данные к табличному виду. - Замените недопустимые символы в заголовках полей. - Отключите или переименуйте поля по необходимости.

4. Завершите настройки и нажмите ОК.

Результат: Статус загрузки отобразится в обработчике задач.

5. Закройте обработчик. Результат: Загруженный файл отобразится в списке коллекций.

При необходимости, вы можете перезапустить задачу.

Загрузка данных из XLSX файлов

  1. Перейдите в раздел Коллекции и нажмите кнопку загрузки над списком коллекций.

  1. В диалоговом окне выберите файл Excel.
  2. В открывшемся окне, введите имя коллекции и подтвердите действие.

Результат: Загруженный файл отобразится в списке коллекций.

Загрузка XLSX файлов позволяет: 1. Загрузить данные из всех листов файла в одну коллекцию. 2. Добавить поле с именем листа при загрузке нескольких листов. 3. Интерпретировать первую строку каждого листа как заголовок. 4. Создать отдельный документ коллекции для каждой строки. 5. Выполнять обработку заголовков полей: - Заменяет недопустимые символы (. и $) на _ в заголовках полей. - Заменяет пустые заголовки на Колонка N, где N — номер колонки. - Добавляет индексы для повторяющихся заголовков.

Вложенные массивы при загрузке интерпретируются как строка.
